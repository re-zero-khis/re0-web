# 『怠惰一闪』

------

「魔女教中最有名的大概是『怠惰』和『强欲』两」

　骑着地龙在旁边并排走着的尤里乌斯如此说道。

　昴光是要在帕特拉修身上坐稳就已经拼尽全力了、与此相对，已经习惯骑乘的尤里乌斯则是一副飒爽的样子。从上面看下来的话，昴大概是一副凄惨的模样，所以并不想跟别并排着走、对于昴的这种纠结究竟尤里乌斯是知道呢还是不知道呢、

「虽然大罪司教的名字是知道的、实际上登在他们的活动资料上的只有这两而已。出现最频繁的是【怠惰】、而说道受害程度的话，则是【强欲】更大」

「虽然是不太好的话题……他们臭名昭著到什么程度呢？」

「太详细的情报我们也不知道」

　对于勉勉强强回应着对话的昴、尤里乌斯捋了捋刘海，看着这边。他斜视着一脸僵硬表情忍耐着摇晃的昴、

「可以把话题进行下去吧？」

「不用在意这边。我老家那边没有乘坐地龙的习惯。啊、帕特拉……你这家伙、不要摇啊！　混蛋蜥蜴！」

「虽然不是说你跟地龙之间的信赖关系不足。地龙好比是你双脚一般的东西、但并不是你的所有物。如果不给予信赖、给它更多自主性的话，两者的步调……」

「别在一边说风凉话。比起这，刚才的话题、可以、继续说下去么」

　对在上面喋喋不休的尤里乌斯顶了回去、昴一边努力调整着与帕特拉修的步调，一边催促着尤里乌斯把话题进行下去。对于昴那不坦率的态度，尤里乌斯呼出一口气、然后竖起一根手指，左右摇着、

「在魔女教的活动中，大罪司教『怠惰』的参战率异常的高。各地的受害事件大概有一半以上都跟『怠惰』这名字有关」

「也就是爱闹事的家伙么」

「从一直担任冲锋陷阵的角色推测、他有着异常的执着心。与『怠惰』之名相对的、是一勤勉的，有能力的家伙。――如果那份热忱不是放到异端行径上的话、究竟会成为一怎样的家伙呢」

　想起了培提尔其乌斯那一心一意地、强烈追求自身的勤勉、以及他的勤勉的那样子。

　可能是正是由于其担当『怠惰』一职、那家伙对正统意义上的『怠惰』十分厌恶的样子。那种反抗心、可能就是在魔女教活动中『怠惰』活动频率异常高的原因。

「但是、都那样频繁地出现的话、还会不了解详细的情报么。骑士团的追击之类的、没有发生过那样的事么」

「基本说来、是很难得到魔女教的活动征兆的。直到事件发生，看到被害现场的才开始意识到那是魔女教干的事、提供情报的基本是那样的。遇上他们还能活下来并不是一件简单的事」

「因此、这次绝对不会让事情发展成那样子」

　对昴的询问，尤里乌斯做出懊悔的回答。而那回答作出强烈否定的是从尤里乌斯另一侧走过来把昴夹在中间的骑乘者――维鲁海鲁姆。

　老眼中蕴含静静的战意、滑出腰间宝剑的剑鞘、

「只要压制住先头部队的话、然后只要正确的以实力对抗就行。我和残存的部队、里卡多大以及尤里乌斯大也在的话、我们是不会处于劣势的」

「那、也对。我也这么认为」

　冷静分析敌我战力差距后、昴对维鲁海鲁姆肯定回应道。

单纯从战力上比较的话、培提尔其乌斯所率领的魔女教的战斗力应该是无法与数上占优势的我方交锋的。除此之外、培提尔其乌斯自身的战斗力也没有太高。

只要接近到他怀中的话、就连外行昴的攻击都无法躲开。如果接近的是维鲁海鲁姆的话、一瞬之间就头落地也是有可能的。

　――攻略那家伙的最大问题果然还是『无形之手』吧。

　关键在于能否在黑雾一般的能伸展到任何地方的手掌中、找准机会接近他。即使说那就是胜负的关键也不为过。然后是、因为能对抗那『无形之手』的只有昴一。

「就是那样的作戦。指望你了、维鲁海鲁姆先生」

「遵命」

　从严肃笑着，作出简短回答的维鲁海鲁姆那里，昴感受到的只有可靠。

　昴不禁有了想将自身一切全部托付与他的冲动、身旁的尤里乌斯则「就是那样」数次同意到、

「我也是、实际上只要到了用剑交锋的阶段的话也不认为自己会输。至今为止一次次让机会从手中溜走，一次次的后悔……这一次、一定不会变成那样的」

　渐渐沸腾的、那是尤里乌斯双眸中静静的愤怒。

　作为近卫骑士团的一员，对于至今为止一次次从手中逃走的魔女教、也是抱有无法平静的感情的罢。

　对此、昴也是同样的意见。

「顺便说下，关于【怠惰】的话题已经说了那么多、另外一位臭名昭著的『强欲』呢？」

「【强欲】和【怠惰】不一样、报告上来的事件并没有那么多。然而、与那名字相关的事件，被害范围实在是过于庞大」

「过于庞大的被害范围……？」

　对昴说到嘴边的话语、尤里乌斯露出认真的表情、陷入思考般闭上了眼睛、

「王国南边的波拉齐亚帝国，帝国中一座因其坚固而被称为城塞都市的【ガークラ】、把那座都市攻陷的就是他。――仅靠一之力」

「攻陷！？　把都市？　一！？」

「士兵必须保持精强――那样的精神在国内流传、在那国家中一兵一卒都是修罗般的。把那样的士兵所守卫的城塞都市攻陷的、就是名为『强欲』的大罪司教一。就连波拉齐亚的英雄『八臂库鲁干』也被打倒」

　对惊讶的张开口的昴、乘胜追击般，维鲁海鲁姆追加道。从口中说出被【强欲】打倒的英雄的名字时、突然皱紧了早已花白的眉头，眼中浮现出乡愁般的感情。

　注意到那样子的昴闭上了嘴、注意到昴的反应，老剑士轻轻摇了摇头、

「与库鲁干是多次以剑交锋的关系。――那还真是好对手。虽然把他八条手臂中的六条砍掉了、与此相对的肚子却被刺了一刀。双方濒死般两败俱伤、最终依旧没有决出胜负就完了」

「居然轻松地说出那样壮绝的话――！」

　说实话，剑鬼现役时代的轶事相当有轻小说的味道，让心动不已。虽然有想继续往下挖掘的想法、但昴还没有神经大条到那样做，毕竟听到从维鲁海鲁姆口中说出的那位好对手被杀的事实。

　这样说来、有关『强欲』的话题对于昴来说也是一沉重的话题。

「怠惰，还有强欲。在此之上还有四……不对、暴食被除掉之后还剩三。还真是任重道远啊」

「――看来好像已经看清楚现实的样子呢」

「仅仅是觉得有点勉强而已啦、可能性还是很大的哦」

　把魔女教的事情在脑海中展开、昴望着远方考虑起了未来的事。

　与『怠惰』培提尔其乌斯的决战、包含对方自己找上门来的意义在内，是无法避免的事态。然后、与培提尔其乌斯决一死战意味着与魔女教的完全对立。

「必然的、与下一大罪冲突的局面也肯定会到来。……就像是引起对方怨恨的展开、一倒下之后，肯定又会有另外一不同的家伙找上门来，这是不良漫画的王道设定」

　相互碰撞、决出胜负就完了、问题是如何实行呢。

　如果说直到其中一方被完全摧毁之前都不会完结的话、只要昴还想继续待在艾米丽娅身旁、与大罪司教的冲突的局面总有一天会到来。

「光是听到有关【强欲】的事就已经足够痛心的了。谁来救救我吧」

「后面的话都还没听完就已经那样消沉的话，话题也很难继续进行下去。――昴、你现在的话只要集中精力到眼前的目标就行了」

「知道的啦。只是因为【怠惰】就在眼前、变得有一点点神经质而已」

　对于尤里乌斯告诫的话语笑着应对、昴重新打起了精神。

　在那之前、尤里乌斯小声道「怠惰……？」，昴虽然听到了、但觉得不是什么大不了的东西，因此忽略掉了。

　夜空开始泛白、黑暗的天空那边，太阳的光芒渐渐开始升了起来。

　已经踏入梅瑟斯领、讨伐队和兽兵团合并后的【对抗魔女教联盟】士气高涨的在骑兽上前进着。

　前进的方向是领主罗兹瓦尔的宅邸？――不是。

　直接前往罗兹瓦尔宅邸、把村里的和屋子里面的艾米丽娅她们全部保护下来、昴曾经想过这也是一办法。

　恐怕从保护他们的角度上来说、那种方式才是风险最低的考虑也是也有的。但是、

「知道对方绝对会来的只有我一、而他们进攻的时机在我们加入护卫的时间点上就已经偏离了。如果变成那样的话、未来的走向就不得而知了」

　武装集团进入宅邸的话、那样培提尔其乌斯可能会重新考虑袭击计划。虽然感觉如果是那狂信者的话，可能也会无视状况的改变勉强执行计划、但是，那已经偏离了昴所知道的未来的样子了。

　攻下白鲸之后、现在联盟的士气非常的高。但是、那种士气在面对不知何时到来的敌面前，究竟能维持到什么时候呢，还是一难题。

　面对敌的恐惧随着时间的流逝会让士气渐渐变弱、此外，一直处于被保护状态下的艾米丽娅和村里的的负担也会相当大。而最终结果则会变得不确定起来。

「既然这样的话、沿着我所知道的未来的形式走下去，取的胜利的方法只有一――只能先发制，一举进攻！」

　恐惧着不知何时到来的对手、看不见未来的做法是不可取的。对于本来不可能会有的，回归死亡后才得到的机会、这是最大限度地有效利用这机会所探寻得到道路。

　这样的话，那方法、可能性是――。

感受着脚下落叶的触感、昴慢慢前进着。

　踩在行走不便的地面上、在昏暗的森林中走着。

　太阳已经爬到相当高的位置、阳光从树木的缝隙之间穿透了过来。一阵风吹过，带走阴湿的空气、昴意识到了自己那过快的心跳以及额头上浮现出来的汗水的冰冷感。

　――现在、昴独身一身前进着。

　在冷风吹过的森林中独自走着的昴、连从街道那里牵出来的帕特拉修都没有带过来，手中也是空无一物、随着脚步不情愿般轻轻摇晃的肩膀，那样前进着的昴实在是相当不可靠的感觉。

「――没有把帕特拉修带过来，总觉得它智商会跟不上此次作战的感觉」

　轻轻笑着、如此抱怨道的昴一副气喘吁吁的样子。

　已经独自一走了相当长的一段距离了。从细长的树木之间穿过、踩断落下的枯枝、小心走下容易打滑的斜波。这种无法称其为道路，但称其为兽道也不太合适的路况，现在只是纯粹阻碍着昴的前进而已。

　像这样行走在这森林里、至今为止已经是第三次了。

　想到第一次和第二次自己手中抱着的是谁的话、现在的自己应该感到相当轻松才对。但是，相比于那时，现在感到脚步更加沉重的原因、是因为面对脚步前方的心情不同吗，还是说因为手中抱着那一位是能够让自己忘记疲劳的存在呢。

「不管、是哪边都好……都不是值的骄傲的事情」

　不管是第一次还是第二次，结果都是无法守护手中抱着的那一位。

　第三次空手而来的现在，绝对要回避之前的悲剧循环。正是由于这目的，才有了现在的行动。

「――――」

　小心回避着看上去有毒的蘑菇、以手边的树干作为支撑，轻轻跳了下来、落地的瞬间、皮肤感受到了空气的变化。

　和白鲸战斗之前的那种紧压皮肤的压迫感不同，现在的空气不是压着皮肤，而是伴随着阴暗的氛围粘了上来的感觉。于是，昴终于无法否认地意识到，自己早已全身冒出冷汗的状态。

「终于来了，……这种、像是在静悄悄的屋子角落、突然意识到蟑螂的存在般的感觉」

　昴与一动不动的黑色物体X无言地互相对视。在不知何时会终结的，孤独的斗争的尽头、是否会产生时间被无限延长般的永恒感呢。

　与那种感觉相近、明显的厌恶感充斥着全身。

　仔细看的话就会发现，不管是左边还是右边都是相似的森林中的景色――但在这里昴确实注意到了，总觉得在哪里见到过的，不可思议的既视感。

　不对、这确实是、曾经来过的地方。

「再这样连正经的道路都没有的地方走着，每次都能到达这里、该说是方向感呢还是说第六感呢、还真是敏锐的让想笑哇」

　还是说、是鼻子比较灵敏的原因么。

　应该取——搜索魔女教专用猎犬的名字比较好么、如果可以的话，还真想辞掉这样的工作。不过这次的话、真心要感谢这能力。

「――特地出来迎接、辛苦各位了」

　向正面的那片昏暗凝视过去、昴如此说道。

　那里存在的是字面意思上毫无情味可言的家伙、对于他的问候更不可能作出回应。对此，自己是再了解不过了。至今才想到、他们究竟是什么东西呢。

　完全就像是没有自身意志的偶一般、连语言、立场都没有，只会一直慢慢蠕动下去、他们自身的意志又在什么地方呢。

「就算是那样问、你们也无法做出回答的吧、魔女教徒」

「――――」

　一眼看去、把昴包围的是与黑暗同化的一身黑色装束的家伙。

　不知何时开始，周围的风声、虫鸣声都早已消失、他们现身时这片土壤早已被重新整顿过一般。因此、与他们的相遇并没有感到吃惊。不如说，他们能够按照预想的出现，心中感到不合时宜的安心感。

「详细的事情跟你们的头目说就行了……总而言之、给我消失吧」

「――――」

「虽然不是很清楚、从地位上来说应该是我在上面的不是么？　拜托了」

　昴抬起手、作出一副要去哪里的样子给他们看。

　随后、黑色装束的家伙向昴表示敬意般低下了头、就以那样的姿势滑进黑暗中溶解消失了。这也是、计划之内的反应。

　虽然心情有点复杂，他们对昴并不带有敌意。只要这边不显示出敌意，或是培提尔其乌斯不作出指示的话，对方袭击过来的可能性是很小的。。

「如果能就那样把他们赶到别的国家去，让他们不要再露脸的话就轻松多了」

　松了一口气、至今为止都能顺利进行是因为还没有失足而已，昴如此想到。

　虽然想过如果培提尔其乌斯那样的指挥系统不存在的话，那样的可能性也并非是零，从现状来看，对那群实体不明的家伙来说在优先度上，昴的位置应该是在培提尔其乌斯之下。目前，最大的毒瘤依旧是那家伙这一点依旧没有变化。

　不管怎么说、能让那群家伙顺利的撤退也是一好消息。作战准备时也考虑过――视情况不同可能会打破原先计划、就在此地展开战场的可能性也是有的。虽然刚刚的接触还能装作一副游刃有余的样子、实际上走的一条还是相当勉强的线。

　对于那样的场面也能无所畏惧地踏入的话，也只能说是过于鲁莽。一贯胆小的自己突然做出如此改变，不禁让觉得可笑。与其说是做好了觉悟、实际上更接近于自暴自弃的感觉，自己分析到。

「啊~ 、还算不错的展开的哦。再一点点就好、拜托了、我的操纵杆」

　确认黑色装束的身影消失之后、昴再次迈开了脚步。

　在只剩下自己的呼吸和脚步声的世界中独自一前进着，这情景让昴产生了、在时间被停止，无限延伸的黑暗中孤独行走的错觉。但是，那样的感慨马上就因为穿过森林，展现在眼前的断崖绝壁而宣告结束。。

　眼前高高耸立的岩壁，像是被森林的巨大抓痕从中间切断一般，巨大的岩石正堵在悬崖正下方的地面上。应该是有什么原理的，使得昴触摸到岩壁后出现的岩壁后面的，前往那冰冷的、黑暗的隐藏洞穴的道路。

　但是、在触摸到岩壁之前、从中走出来迎接的影子先一步注意到了昴这边。

「――已等候多时。被宠爱的信徒哟」

　站在那里的是摊开双手、浮现出喜悦之情的男。

　他有着瘦削的脸颊，深陷的眼眶，深绿色的头发也毫无生气可言、黑色法衣下隐约可窥见的手脚如枯枝般细弱。年龄上来看大约三十五、六的样子，但从他整体传递出来的虚弱感来看，就算说是五十多岁也不会让惊讶。

　但是、依旧没变的是那双盯视着这边，闪耀着疯狂的光芒的眼睛。那双寄宿着疯狂的眼睛仿佛要将昴舔舐殆尽般投来了视线。

「我是魔女教、大罪司教『怠惰』担当。培提尔其乌斯・罗马尼……空提！」
