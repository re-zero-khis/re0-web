**WEB版 与 文库版 的区别**：

- WEB版相当于草稿，文库版在此基础上进行了大量修缮（甚至删减）
- 文库版无论剧情精彩度、人物刻画深度、还是逻辑严谨度都比WEB版要好
- 推荐先看文库版，WEB版建议用于了解最新剧情
